spamhaus_sbl_parser
состоит из двух частей (сам парсер + веб интерфейс к базе)

сам парсер (python или php на выбор):
 - раз в 30 минут (по крону) чекает список SBL
(http://spamhaus.org/sbl/latest/)
 - чекат через TOR чтобы менять IP
 - если попадает на cloudflare, или еще какую блокировку, то чекает
повтороно, с другого IP (нужно сменить цепочку TOR)
 - пишет подробный лог в базу
 - пишет в базу каждый листинг + изменение его статуса (sbl_out)
 - в качестве базы используеют любую встриваемую sql/nosql базу

веб интерфейс (на php+yii, python+django, или чистом php/python):
 - постраничный вывод всех листингв
 - если кликнуть на листинг переходим на его текст
 - удобный полнотекстовый поиск по листингам

немного про TOR, чтобы не тратить время на гугл:
- нужно его установить (apt-get install tor)
- запустить (/etc/init.d/tor start)
- приконнектится к нему (socks 127.0.0.1:9050)
- менять цепочки (можно по разному, но проще всего /etc/init.d/tor reload)

если нужно-выдадим vps для тестов
